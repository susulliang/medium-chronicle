---

The rapid advancement of new media technology carries a fatal flaw: digital art suffers from accelerated obsolescence. Traditional museums, built to preserve canvas and stone for centuries, are ill-equipped to maintain software that requires legacy hardware or deprecated code libraries to function.

This archival crisis is particularly acute for interactive installations. When an artwork relies on an outdated sensor, an unsupported operating system, or a live API that has been shut down, the work fundamentally dies. Documentation—video recordings, photographs, and written descriptions—remains the only proof that the experience ever occurred.

Preservationists are pioneering emulation strategies, building virtual machines that mimic ancient hardware environments. Yet, the question remains: if an interactive artwork is run on an emulator fifty years after its creation, devoid of its original cultural and physical context, is it still the same piece of art?
